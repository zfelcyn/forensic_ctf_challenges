# Gone But Not Forgotten

## Purpose
This challenge teaches deleted-file recovery from a raw disk image. Solvers can get oriented with `strings`, then move to a filesystem-focused workflow in tools like Autopsy or Foremost to recover the deleted note containing the flag.

## Contents
- `assets/laptop.img`: Static disk image copied into each generated bundle
- `generate.py`: Bundle generator that writes the solver-facing files
- `challenge.json`: Portal manifest and flag validation metadata
